# SEO Configuration Guide

This document outlines the SEO optimizations implemented for BetterPerformance to improve search engine visibility, especially for keywords like:
- Windows 10 tweaks
- Windows 11 tweaks
- Twix for Windows
- Better performance Windows
- Windows performance optimization

## Environment Variables

Add these to your `.env.local` file:

```env
# Site URL (optional - defaults to betterperformance.vercel.app)
NEXT_PUBLIC_SITE_URL=https://betterperformance.vercel.app
```

**Note:** The application defaults to `https://betterperformance.vercel.app` if `NEXT_PUBLIC_SITE_URL` is not set.
